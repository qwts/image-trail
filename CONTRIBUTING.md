# Contributing to Image Trail

The canonical contributor and agent workflow guide lives in the GitHub wiki:

- [Contributing](https://github.com/qwtm/image-trail/wiki/Contributing)

Read the wiki guide before starting tracked work. When workflow,
documentation, issue-claim, branch, PR, or validation rules change, update the
wiki page instead of expanding this repo stub.

Source-adjacent docs still live in this repository when they need to version
with code, such as acceptance tests, architecture notes, schema/migration notes,
and CI automation policy.
